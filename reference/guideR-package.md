# guideR: Miscellaneous Statistical Functions Used in 'guide-R'

Companion package for the manual 'guide-R : Guide pour l’analyse de
données d’enquêtes avec R' available at
<https://larmarange.github.io/guide-R/>. 'guideR' implements
miscellaneous functions introduced in 'guide-R' to facilitate
statistical analysis and manipulation of survey data.
