# Fit indices for dominance analysis of survey-weighted GLM

Provides support of
[`survey::svyglm()`](https://rdrr.io/pkg/survey/man/svyglm.html) for
[`dominanceanalysis::dominanceAnalysis()`](https://rdrr.io/pkg/dominanceanalysis/man/dominanceAnalysis.html).

## Usage

``` r
da.svyglm.fit(original.model, newdata = NULL, ...)
```

## Arguments

- original.model:

  Original fitted model

- newdata:

  Data used in update statement

- ...:

  ignored
