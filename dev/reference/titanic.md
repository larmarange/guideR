# Titanic data set in long format

This `titanic` dataset is equivalent to
`datasets::Titanic |> dplyr::as_tibble() |> tidyr::uncount(n)`.

## Usage

``` r
titanic
```

## Format

An object of class `tbl_df` (inherits from `tbl`, `data.frame`) with
2201 rows and 4 columns.
